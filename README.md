# AI Artifact Manager 🎬✨

生成AIのプロンプトと出力されたアーティファクト（画像、音声、動画、ドキュメント、URL、アプリ）をポップに管理するツールです。

## 🚀 使い方

### 1. 準備
Pythonがインストールされていることを確認してください。

### 2. 起動方法
`run.bat` をダブルクリックして実行してください。
- 初回起動時に必要なライブラリが自動的にインストールされます。📦
- ブラウザが立ち上がり、管理画面が表示されます。✨

### 3. アーティファクトの追加
- **New Job ✨** ボタンから追加できます。
- ファイルパスは **Browse...** ボタンで簡単に選択できます。
- エクスプローラー画面では、名前や作成日時でソート可能です。📅

---

## 📦 配布用EXEの作成

自分専用のEXEファイルを作りたい場合は、以下の手順を行ってください。

1. `build.bat` をダブルクリックします。
2. しばらく待つと `dist\AIArtifactManager` フォルダが生成されます。
3. このフォルダをまるごと配布することで、Pythonがない環境でも動作します。🚀
