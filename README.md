# AI Artifact Manager 🎬✨

生成AIのプロンプトと出力されたアーティファクト（画像、音声、動画、ドキュメント、URL、アプリ）をポップに、そしてスマートに管理する次世代のアーティファクト管理ツールです。🚀

---

## 🌟 特徴

- **マルチメディア対応**: 画像、音声、動画、ドキュメント、URL、そして実行可能なアプリ（バッチファイルなど）を一元管理。🎬🎵
- **グローバルオーディオプレイヤー**: 画面下部に常駐するスタイリッシュなプレイヤー。どこからでも音楽をコントロールでき、アルバムアートも自動抽出！🖼️🔊
- **高度なプレイリスト管理**: ドラッグ＆ドロップによる並び替え、シャッフル再生、曲の予約（キュー追加）機能を搭載。🔀🤝
- **状態の自動保存**: 音量設定やプレイリストの内容を自動保存。リロードしてもすぐに続きから再開できます。💾✨
- **インAppエクスプローラー**: 作成日時や名前でソート可能な、使いやすいファイル選択画面。📅🔍
- **アプリ実行機能**: バッチファイルやURLをボタン一つで実行・起動。⚡

---

## 🚀 使い方

### 1. 準備
Pythonがインストールされていることを確認してください。

### 2. 起動方法
`run.bat` をダブルクリックして実行してください。
- 初回起動時に必要なライブラリが自動的にインストールされます。📦
- ブラウザが自動的に立ち上がり、美しい管理画面が表示されます。✨

### 3. アーティファクトの管理
- **New Task ✨** ボタンからアーティファクトを追加。
- オーディオカードの **▶️** ボタンで即座に再生、**➕** ボタンでプレイリストの最後に予約できます。
- 画面左上の **Play All 🎵** ボタンで、現在フィルタリングされているすべての曲を連続再生します。

---

## 📦 配布用EXEの作成

自分専用のEXEファイルを作りたい、または仲間に配布したい場合は以下の手順で行ってください。

1. `build.bat` をダブルクリックします。
2. しばらく待つと `dist\AIArtifactManager` フォルダが生成されます。
3. このフォルダをまるごとコピーするだけで、Pythonがない環境でも動作します。🚀🛸💨

## 🛠️ 技術スタック

- **Backend**: Python (Eel, Bottle, Mutagen)
- **Frontend**: HTML5, Vanilla CSS (Glassmorphism), JavaScript
- **State Management**: LocalStorage for Persistence
- **Communication**: Bidirectional IPC via Eel
